## Description

NestJS based api for dog tinder application.

## Prerequirments

Docker daemon must be installed on your machine to run this application in local mode

## Installation

```bash
$ npm ci
```

## Running the app

```bash
# docker
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
